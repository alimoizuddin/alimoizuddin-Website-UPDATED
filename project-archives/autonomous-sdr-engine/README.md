# Autonomous SDR Engine & Executive Ghostwriting Pipeline

An n8n automation that turns lead inputs into researched, quality-controlled outreach drafts. It combines public signal scraping, filtering, QA logic, and identity-aware draft generation.

## Included

- `screenshots/` - full architecture, dual-agent logic, API throttling, CRM output, and frontend/backend proof screens.
- `docs/` - project explainer documents.
- `demo/Autonomous SDR Engine_ Executive Ghostwriting Pipeline.mp4` - demo recording.
- `workflows/autonomous-sdr-engine.sanitized.json` - sanitized n8n workflow export.

## Sanitization

The workflow export has credential blocks removed. The Apify API token and webhook IDs were redacted before publishing.
